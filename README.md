## 🚀 **QUICK START**

### 1. **Clone the Repository**
```bash
git clone https://github.com/RAJSARASWATI-JATAV/ultimate-creator-mega-suite.git
cd ultimate-creator-mega-suite
```

### 2. **Setup Script (Linux/Mac)**
```bash
chmod +x tools/automation/setup.sh
./tools/automation/setup.sh
```

### 3. **Setup Script (Windows)**
```cmd
tools\automation\setup.bat
```

### 4. **Manual Setup**
```bash
# Install Python dependencies
pip install -r requirements.txt

# Install Node.js dependencies
npm install

# Setup database
python tools/database/setup_db.py

# Start the application
python src/backend/python/main.py
```
